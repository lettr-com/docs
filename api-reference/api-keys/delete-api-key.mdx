# Delete API Key

Revoke an API key. This is immediate and permanent.

<ParamField path="id" type="string" required>
  The ID of the API key to delete.
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl -X DELETE https://api.lettr.dev/v1/api-keys/key_123abc \
  -H "Authorization: Bearer le_xxxxxxxxxxxx"
```

```javascript Node.js
await lettr.apiKeys.delete('key_123abc');
```

```php PHP
$lettr->apiKeys->delete('key_123abc');
```
</CodeGroup>

## Response

```json
{
  "id": "key_123abc",
  "object": "api_key",
  "deleted": true
}
```

<Warning>
  Deleting an API key is immediate and permanent. Any applications using this key will immediately lose access.
</Warning>

## Errors

| Code | Description |
|------|-------------|
| `not_found` | API key with specified ID not found |
| `cannot_delete_self` | Cannot delete the API key used for this request |
