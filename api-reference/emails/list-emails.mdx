# List Emails

Retrieve a paginated list of email events with optional filtering.

<ParamField query="per_page" type="integer" default="10">
  Number of emails to return per page. Maximum 100.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="from_date" type="string">
  Filter emails created after this ISO 8601 timestamp.
</ParamField>

<ParamField query="to_date" type="string">
  Filter emails created before this ISO 8601 timestamp.
</ParamField>

## Response Fields

### Email Event Object

| Field | Type | Description |
|-------|------|-------------|
| `request_id` | string | The transmission/request identifier |
| `message_id` | string | The unique message identifier |
| `event_id` | string | Unique identifier for this event |
| `subject` | string | Email subject line |
| `friendly_from` | string | Sender display name and address |
| `sending_domain` | string | Domain used for sending |
| `timestamp` | string | ISO 8601 timestamp of the event |
| `click_tracking` | boolean | Whether click tracking is enabled |
| `open_tracking` | boolean | Whether open tracking is enabled |
| `rcpt_to` | string | Recipient email address |
| `raw_rcpt_to` | string | Raw recipient with display name |
| `recipient_domain` | string | Domain of the recipient |
| `mailbox_provider` | string | Detected mailbox provider (e.g., Gmail, Gsuite, Outlook) |
| `mailbox_provider_region` | string | Geographic region or category of the mailbox provider |
| `rcpt_meta` | array \| object | Custom metadata attached to the recipient |
| `sending_ip` | string | IP address used for sending |
| `transactional` | boolean | Whether the email was marked as transactional |
| `msg_size` | string | Size of the message in bytes |
| `injection_time` | string | ISO 8601 timestamp when email was injected |

### Pagination Object

| Field | Type | Description |
|-------|------|-------------|
| `total_pages` | integer | Total number of pages available |
| `current_page` | integer | Current page number |
| `per_page` | integer | Number of results per page |

## Pagination Example

```javascript
// Fetch all emails using pagination
let allEmails = [];
let currentPage = 1;
let totalPages = 1;

do {
  const response = await lettr.emails.list({
    per_page: 100,
    page: currentPage
  });
  
  allEmails = allEmails.concat(response.results);
  totalPages = response.pagination.total_pages;
  currentPage++;
} while (currentPage <= totalPages);
```
