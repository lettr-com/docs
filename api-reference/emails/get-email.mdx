# Get Email

Retrieve details of a specific email by its ID.

<ParamField path="id" type="string" required>
  The ID of the email to retrieve.
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl https://api.lettr.dev/v1/emails/email_123abc \
  -H "Authorization: Bearer le_xxxxxxxxxxxx"
```

```javascript Node.js
const email = await lettr.emails.get('email_123abc');
```

```php PHP
$email = $lettr->emails->get('email_123abc');
```

```python Python
email = lettr.emails.get('email_123abc')
```
</CodeGroup>

## Email Status

| Status | Description |
|--------|-------------|
| `queued` | Email is queued for sending |
| `sent` | Email has been sent |
| `delivered` | Email was delivered |
| `bounced` | Email bounced |
| `failed` | Delivery failed |
| `scheduled` | Email is scheduled for future delivery |
| `cancelled` | Scheduled email was cancelled |
| `suppressed` | Recipient was suppressed |

## Errors

| Code | Description |
|------|-------------|
| `not_found` | Email with specified ID not found |
