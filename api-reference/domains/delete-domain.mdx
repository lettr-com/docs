# Delete Domain

Remove a domain from your account.

<ParamField path="domain" type="string" required>
  The domain name to delete.
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl -X DELETE https://app.uselettr.com/api/domains/mail.example.com \
  -H "Authorization: Bearer le_xxxxxxxxxxxx"
```

```javascript Node.js
await lettr.domains.delete('mail.example.com');
```

```php PHP
$lettr->domains->delete('mail.example.com');
```

```python Python
lettr.domains.delete('mail.example.com')
```
</CodeGroup>

## Response

Returns `204 No Content` on successful deletion.

<Warning>
  Deleting a domain is permanent. You will no longer be able to send emails from addresses on this domain.
</Warning>

## Errors

| Code | Description |
|------|-------------|
| `not_found` | Domain not found in your account |
