# Get Domain

Retrieve details of a specific domain by its ID.

<ParamField path="id" type="string" required>
  The ID of the domain to retrieve.
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl https://api.lettr.dev/v1/domains/dom_123abc \
  -H "Authorization: Bearer le_xxxxxxxxxxxx"
```

```javascript Node.js
const domain = await lettr.domains.get('dom_123abc');
```

```php PHP
$domain = $lettr->domains->get('dom_123abc');
```
</CodeGroup>

## Domain Status

| Status | Description |
|--------|-------------|
| `pending` | Waiting for DNS verification |
| `verified` | All DNS records verified |
| `failed` | Verification failed |
