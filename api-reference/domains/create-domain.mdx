# Create Domain

Add a new domain for sending emails.

<ParamField body="name" type="string" required>
  The domain name (e.g., `example.com`).
</ParamField>

<ParamField body="region" type="string" default="us">
  Data region: `us` or `eu`.
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl -X POST https://api.lettr.dev/v1/domains \
  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "example.com"
  }'
```

```javascript Node.js
const domain = await lettr.domains.create({
  name: 'example.com'
});
```

```php PHP
$domain = $lettr->domains->create([
    'name' => 'example.com'
]);
```
</CodeGroup>

## Next Steps

After creating a domain:

1. Add the DNS records shown in the response to your domain
2. Wait for DNS propagation (usually 15 minutes to 48 hours)
3. Call the [Verify Domain](/api-reference/domains/verify-domain) endpoint
