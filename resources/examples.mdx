# Examples

Learn from complete code examples for sending emails with the Lettr API.

## Send a Simple Email

```bash
curl -X POST https://app.lettr.com/api/emails \
  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": ["user@example.com"],
    "subject": "Hello World",
    "html": "<p>Welcome to Lettr!</p>"
  }'
```

```bash
  }'
```

## Send with CC and BCC

```bash

  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "team@yourdomain.com",
    "to": ["recipient@example.com"],

    "bcc": ["archive@example.com"],
    "subject": "Project Update",
    "html": "<p>Here is the latest project update.</p>"
  }'
```

## Send with Custom Options

```bash
curl -X POST https://app.lettr.com/api/emails \
  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "notifications@yourdomain.com",
    "from_name": "My App",
    "to": ["user@example.com"],
    "subject": "Password Reset",
    "html": "<p>Click the link to reset your password.</p>",
    "reply_to": "support@yourdomain.com",
    "options": {
      "click_tracking": false,
      "open_tracking": false,
      "transactional": true
    }
  }'
```

## Send with Metadata and Campaign

```bash
curl -X POST https://app.lettr.com/api/emails \
  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "marketing@yourdomain.com",
    "to": ["subscriber@example.com"],
    "subject": "Special Offer",
    "html": "<p>Check out our special offer!</p>",
    "campaign_id": "spring-sale-2024",
    "metadata": {
      "user_id": "12345",
      "source": "website"
    }
  }'
```

## Send with Variable Substitution

```bash
curl -X POST https://app.lettr.com/api/emails \
  -H "Authorization: Bearer le_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "welcome@yourdomain.com",
    "to": ["newuser@example.com"],
    "subject": "Welcome, {{name}}!",
    "html": "<p>Hello {{name}}, welcome to our platform!</p>",
    "substitution_data": {
      "name": "John"
    }
  }'
```
