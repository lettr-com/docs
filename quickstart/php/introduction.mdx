# PHP Quickstart

Integrate Lettr into your PHP application to send transactional emails. Choose the guide that matches your framework.

## Prerequisites

Before you begin, make sure you have:

1. A Lettr account — [sign up at app.lettr.com](https://app.lettr.com)
2. An [API key](/learn/api-keys/introduction) created in your dashboard
3. A [verified sending domain](/learn/domains/introduction)
4. PHP 8.1 or higher

## Direct API Usage

For plain PHP without a framework, you can send emails directly via the REST API:

```php
$apiKey = 'lttr_xxxxxxxxxxxx';

$data = [
    'from' => 'you@yourdomain.com',
    'to' => ['recipient@example.com'],
    'subject' => 'Hello from Lettr',
    'html' => '<p>Your email content here.</p>'
];

$ch = curl_init('https://app.lettr.com/api/emails');
curl_setopt_array($ch, [
    CURLOPT_POST => true,
    CURLOPT_POSTFIELDS => json_encode($data),
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
        'Accept: application/json',
    ],
    CURLOPT_RETURNTRANSFER => true,
]);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

echo $response;
```

For the full API reference, see the [Send Email endpoint](/api-reference/emails/send-email).
