## 1. Install

Get the Lettr Python SDK and Flask.

```bash
pip install lettr flask
```

## 2. Send email using HTML

The easiest way to send an email is by using the `html` parameter. Create a route and call the Lettr client.

```python
import os
import lettr
from flask import Flask, jsonify

client = lettr.Lettr(os.environ["LETTR_API_KEY"])

app = Flask(__name__)


@app.route("/")
def index():
    response = client.emails.send(
        from_email="Acme <onboarding@yourdomain.com>",
        to=["delivered@example.com"],
        subject="hello world",
        html="<strong>it works!</strong>",
    )
    return jsonify({"request_id": response.request_id, "accepted": response.accepted})


if __name__ == "__main__":
    app.run()
```

Run the app (with `LETTR_API_KEY` set), then open `http://127.0.0.1:5000/` to trigger the send.
