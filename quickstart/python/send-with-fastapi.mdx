## 1. Install

Get the Lettr Python SDK and FastAPI.

```bash
pip install lettr fastapi uvicorn
```

## 2. Send email using HTML

The easiest way to send an email is by using the `html` parameter. Create an endpoint and call the Lettr client.

```python
import os
from typing import Any, Dict

import lettr
from fastapi import FastAPI

client = lettr.Lettr(os.environ["LETTR_API_KEY"])

app = FastAPI()


@app.post("/send")
def send_mail() -> Dict[str, Any]:
    response = client.emails.send(
        from_email="Acme <onboarding@yourdomain.com>",
        to=["delivered@example.com"],
        subject="hello world",
        html="<strong>it works!</strong>",
    )
    return {"request_id": response.request_id, "accepted": response.accepted}
```

Run the app with `uvicorn main:app --reload` (and `LETTR_API_KEY` set), then `POST /send` to trigger the send.
